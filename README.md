# Diziel GitHub Pages Ready

هذه نسخة Frontend static كاملة وجاهزة للرفع على GitHub Pages.

## الصفحات
- Login
- Register
- Dashboard
- Orders
- Drivers
- Vehicles
- Reports
- Settings

## المميزات
- تعمل بدون Backend
- تحفظ البيانات داخل localStorage
- تدعم GitHub Pages مباشرة
- تستخدم relative paths
- يوجد ملف 404.html لإعادة التوجيه

## التشغيل المحلي
افتح `index.html` مباشرة أو استخدم Live Server.

## الرفع على GitHub Pages
1. ارفع كل الملفات كما هي إلى repository جديد.
2. افتح Settings.
3. افتح Pages.
4. اختر Deploy from a branch.
5. اختر branch = main.
6. اختر folder = /(root).
7. احفظ.

## تسجيل الدخول
اكتب أي Email و Password للتجربة.
